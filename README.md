# Gobang
五子棋

## 背景
下午上人工智能的时候看旁边同学在玩五子棋，于是开始想实现的逻辑。尽管大二的时候就想写一个试试，但是现在才开始动手ORZ。欢迎fork。

## 实现
目前只实现了五子连珠的基本逻辑，可以在控制台进行双人下棋。
